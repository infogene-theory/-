# 📋 **社区规范**

为了维护一个健康、开放、友好的社区环境，我们制定了以下社区规范，希望所有社区成员共同遵守。

## 🤝 **核心价值观**

### 1. 开放包容
- 尊重不同观点和文化背景
- 欢迎来自不同领域的专家和爱好者
- 鼓励学术讨论和思想碰撞
- 不排斥新思想和新方法

### 2. 科学严谨
- 保持科学态度，尊重事实和证据
- 鼓励基于数据和逻辑的讨论
- 避免主观臆断和情绪化表达
- 接受并尊重不同的学术观点

### 3. 合作共赢
- 鼓励社区成员之间的合作
- 分享知识和资源
- 互相支持和帮助
- 共同推动IGT理论的发展

### 4. 诚信负责
- 遵守学术规范和伦理准则
- 尊重知识产权
- 对自己的言论负责
- 不发布虚假信息

## 📌 **行为准则**

### 1. 尊重他人
- 不进行人身攻击或辱骂
- 不歧视任何个人或群体
- 尊重他人的隐私
- 不泄露他人的个人信息

### 2. 理性讨论
- 基于事实和逻辑进行讨论
- 避免情绪化表达
- 不使用攻击性语言
- 接受不同观点

### 3. 保持专业性
- 讨论内容与IGT理论相关
- 保持学术严谨性
- 引用可靠来源
- 避免发布无关内容

### 4. 遵守法律法规
- 不发布违法违规内容
- 不传播有害信息
- 遵守知识产权法规
- 尊重他人的合法权益

## 🚫 **禁止行为**

### 1. 人身攻击
- 对个人进行辱骂、诽谤或威胁
- 恶意贬低他人的观点或贡献
- 进行人身攻击或人身威胁
- 发布侮辱性内容

### 2. 虚假信息
- 发布虚假或误导性信息
- 传播未经证实的谣言
- 伪造数据或证据
- 夸大或扭曲事实

### 3. 垃圾信息
- 发布与IGT理论无关的内容
- 重复发布相同内容
- 发布广告或推广信息
- 发布恶意链接

### 4. 侵权行为
- 侵犯他人的知识产权
- 未经授权使用他人的作品
- 抄袭或剽窃他人的研究成果
- 侵犯他人的隐私

## ⚠️ **违规处理**

### 1. 处理流程
- **第一次违规**：警告并删除违规内容
- **第二次违规**：限制发言权限
- **第三次违规**：暂时封禁账号
- **严重违规**：永久封禁账号

### 2. 申诉机制
- 如果您认为处理不当，可以提出申诉
- 发送邮件至contact@imeta.space
- 提供详细的申诉理由和证据
- 核心团队将在3个工作日内回复

## 📢 **社区监督**

我们鼓励社区成员参与社区监督，包括：
- 举报违规行为
- 提供改进建议
- 参与社区管理
- 维护社区秩序

## 📌 **建议和反馈**

如果您对社区规范有任何建议或反馈，欢迎通过以下方式联系我们：
- 在GitHub Discussions中发帖
- 在Discord社群中讨论
- 发送邮件至contact@imeta.space

我们会定期审查和更新社区规范，确保其适应社区的发展和变化。

## 🌟 **共建美好社区**

元衍智库的成功离不开每一位社区成员的贡献和支持。让我们共同遵守社区规范，营造一个开放、包容、科学、严谨的社区环境，共同推动IGT理论的发展和应用！
