# IGT空间调频平面设计稿规范文档

## 设计总览

### 设计原则
基于IGT-RVS机制的视觉秩序理论，通过精确的VOI（Visual Order Index）设计，实现空间认知频率的精准调节。所有设计稿严格按照300dpi印刷精度和CMYK色彩模式制作，确保最佳的视觉传达效果。

### 设计目标
- **认知共振**：通过视觉元素与脑波频率的协同
- **秩序构建**：提升空间O_freq（秩序度）≥0.7
- **情绪调节**：降低CV（变异系数）≤0.15
- **美学统一**：现代简约与功能性的完美结合

## VOI等级设计规范

### VOI1级设计规范（高放松场景）

#### 色彩配置
**背景色彩**：
- 深蓝底：C100 M80 Y20 K30
- 深紫渐变：C80 M100 Y30 K40 → C60 M80 Y20 K20
- 浅紫底：C30 M40 Y10 K5

**文字色彩**：
- 主文字：K15（浅灰）
- 辅助文字：K25（中灰）
- 强调文字：C20 M15 Y10 K10

**留白比例**：≥60%
**色彩饱和度**：≤40%

#### 字体设计
**主标题字体**：
- 字体类型：圆润曲线字体
- 字体大小：48-60pt
- 字重：中等（Medium）
- 字间距：标准+5%
- 行间距：140%

**正文字体**：
- 字体类型：柔和无衬线字体
- 字体大小：24-32pt
- 字重：正常（Regular）
- 字间距：标准
- 行间距：150%

#### 版式设计
**布局结构**：
- 上下结构：标题在上，内容在下
- 左右结构：左文右图或左图右文
- 中心对称：文字居中排列

**视觉元素**：
- 曲线装饰：柔和曲线线条
- 渐变效果：平滑色彩过渡
- 光影处理：柔和阴影效果

## 场景专项设计

### 睡眠场景设计（S-θ-SL）

#### 设计要素
**核心标语**：
- SL01：呼吸调频，思绪归序
- SL02：身心共振，美梦自来
- SL03：频率自洽，夜眠无扰

**脑波对应**：θ波（4-7Hz）
**设计目标**：情绪平复、深度睡眠
**O_freq目标**：≥0.8
**CV目标**：≤0.1

#### 专属设计元素
**图形元素**：
- 呼吸韵律曲线
- 月相变化图案
- 星空渐变背景
- 水波纹效果

**色彩心理**：
- 深蓝：宁静、安详
- 深紫：神秘、深度
- 银色：纯净、平和

### 家庭场景设计（S-α-FM）

#### 设计要素
**核心标语**：
- FM01：言行共振，家序自安
- FM02：共识为基，相处无扰
- FM03：频率同频，家和业兴

**脑波对应**：α波（8-12Hz）
**设计目标**：家庭协同、情绪共鸣
**O_freq目标**：≥0.75
**CV目标**：≤0.12

#### 专属设计元素
**图形元素**：
- 家庭树图案
- 手牵手剪影
- 温暖光晕效果
- 和谐几何图案

**色彩心理**：
- 暖黄：温暖、亲情
- 橙红：活力、热情
- 原木：自然、温馨

### 办公场景设计（S-β-WR）

#### 设计要素
**核心标语**：
- WR01：目标同频，协作高效
- WR02：专注当下，认知有序
- WR03：决策有序，执行有力

**脑波对应**：β波（13-30Hz）
**设计目标**：专注高效、决策清晰
**O_freq目标**：≥0.7
**CV目标**：≤0.15

#### 专属设计元素
**图形元素**：
- 箭头指向图案
- 效率齿轮图标

- 流线型几何

**色彩心理**：
- 浅蓝：理性、效率
- 冷灰：专业、稳重
- 银灰：科技、精密

## 技术制作规范

### 文件格式规范
**原始设计文件**：
- 格式：AI（Adobe Illustrator）
- 版本：CC 2020或更高
- 色彩模式：CMYK
- 分辨率：300dpi
- 包含图层：可编辑

**印刷输出文件**：
- 格式：PDF/X-4:2010
- 色彩模式：CMYK
- 分辨率：300dpi
- 压缩：高质量
- 包含出血：3mm

**预览展示文件**：
- 格式：JPG/JPEG
- 色彩模式：sRGB
- 分辨率：150dpi
- 质量：80%
- 尺寸：适合屏幕显示

### 印刷技术要求

#### 色彩管理
**色彩模式**：CMYK四色印刷
**色彩标准**：ISO Coated v2 (ECI)
**色彩偏差**：ΔE≤3.0
**专色印刷**：可选Pantone配色

#### 精度要求
**印刷精度**：≥2400dpi
**网点扩大**：≤15%
**套印精度**：≤0.1mm
**线条最细**：0.1mm

#### 材料规格
**基材材质**：哑光PVC
**厚度规格**：0.2-0.3mm
**表面处理**：抗UV涂层
**背面处理**：可移除胶层

### 尺寸标注规范

#### 标准尺寸
**成品尺寸**：600×900mm
**出血尺寸**：606×906mm（各边+3mm）
**安全距离**：距边缘≥20mm
**折线位置**：按实际需要标注

#### 标注要求
**标注单位**：毫米（mm）
**标注精度**：±0.1mm
**标注字体**：Arial 10pt
**标注颜色**：K100（黑色）
