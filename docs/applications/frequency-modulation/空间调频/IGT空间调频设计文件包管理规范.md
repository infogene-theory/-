## 文件命名规范

### 命名规则
```
项目代码_文件类别_具体内容_版本号.文件扩展名

示例：
IGT_RVS_物料清单_设备类_v2.1.xlsx
IGT_RVS_设计规范_VOII级标准_v1.2.pdf
IGT_RVS_效果图_睡眠空间_01_v2.1.png
```

### 版本管理
```
版本号格式：主版本号.次版本号
- 主版本号：重大修改时递增
- 次版本号：小幅修改时递增
- 初始版本：v1.0
- 当前版本：v2.1
```

### 日期标记
```
文件创建日期：YYYY-MM-DD格式
文件修改日期：YYYY-MM-DD格式
文件生效日期：YYYY-MM-DD格式
```

## 文件格式规范

### 设计文件格式
```
源文件：AI、PSD、DWG（可编辑格式）
印刷文件：PDF/X-4、TIFF（高质量格式）
预览文件：JPG、PNG（中等质量格式）
文档文件：DOCX、MD（可编辑格式）
表格文件：XLSX（可计算格式）
```

### 质量标准
```
图像分辨率：300dpi（印刷用）
图像分辨率：150dpi（预览用）
色彩模式：CMYK（印刷用）
色彩模式：sRGB（屏幕用）
压缩质量：80%（预览用）
```
