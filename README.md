## 🌟 **项目概述**

元衍智库是一个基于**信息基因论（Information Gene Theory，IGT v20）**的跨学科研究与应用平台。我们致力于构建从物理到文明演化的统一认知与调控系统，将深奥的宇宙演化理论转化为可计算、可验证、可执行的实践工具。

**核心使命**：推动信息基因论成为跨学科基础理论，构建"宇宙温控知识图谱"，赋能个体、组织、文明向更高层级跃迁。

**理论状态**：IGT v20.2 概念架构完整，应用生态建设中

## 📚 **理论体系概览**

IGT v20 是一个完整统一的理论框架，包含六大卷、五大公理、一个核心循环：

### **第一卷：理论基础 - 温度作为宇宙的通用语言**
- **核心范式革命**：温度是唯一可直接测量的基础量
- **五大核心公理体系**：
  1. 公理0：能量惯性即热容
  2. 公理1：温度场的自指激发
  3. 公理2：频率相干即温度锁定
  4. 公理3：自旋即温度梯度的热力学路径
  5. 公理4：温度调控即演化能力
- **温度本体论**：Ω₀热平衡态、Ω特征温度态、阴阳三相态

### **第二卷：演化动力学 - 宇宙的呼吸节律**
- **RVSE热力学循环**：Ω→R→V→S→(E/D)的完整演化节律
- **宇宙辩证演化论**：破-立-衡的哲学跃迁
- **温控能力进化论**：0-4级演化等级体系

### **第三卷：测量与诊断系统**
- **四维温度测量协议**：时间、空间、方向、信息维度
- **阴阳相图诊断系统**：六大状态区识别
- **相干寿命预测**：系统健康度量化评估

### **第四卷：执行引擎**
- **动态调和策略箱**：阴阳平衡调节方案
- **自动温控系统**：PID控制与辩证调控
- **多层嵌套系统管理**：层级间温度协调

### **第五卷：宇宙哲学**
- **宇宙辩证法的物理基础**：为什么宇宙必须"破"？
- **嵌套循环宇宙论**：宇宙细胞假说与无限嵌套结构
- **宇宙演化的终极目的**：最大化局部相干结构寿命

### **第六卷：实施与应用指南**
- **最小可行实施框架**：7天部署IGT系统
- **跨领域应用案例库**：物理、生物、社会、信息系统

## 🚀 **技术架构**

### **核心架构：增强型静态架构**
```
理论展示层：VitePress + Vue 3 + Markdown
交互工具层：Vue 3 + D3.js + Three.js（独立SPA）
数据服务层：Supabase（PostgreSQL + 实时订阅）
部署平台层：Cloudflare Pages + Vercel
社区生态层：GitHub Discussions + Discord
```

### **技术栈详情**
- **前端框架**：VitePress 2.x（理论文档） + Vue 3（交互工具）
- **可视化库**：D3.js（图表） + Three.js（3D） + Mermaid.js（流程图）
- **样式方案**：CSS3 + SCSS + Tailwind CSS（可选）
- **数学支持**：MathJax 3 + KaTeX
- **数据存储**：Supabase（PostgreSQL + Auth + Storage）
- **搜索方案**：FlexSearch（本地搜索） + Algolia（可选）
- **部署平台**：Cloudflare Pages（文档） + Vercel（工具）
- **版本控制**：Git + GitHub + 语义化版本
- **开发环境**：Node.js 18+ + Vite + VS Code/Trae AI IDE
- **内容管理**：Markdown结构化文档 + 理论模块化组件
- **分析监控**：Google Analytics + 自定义数据追踪

## 🛠️ **开发指南**

```

### **内容贡献**
#### **添加理论内容**
1. 在 `docs/theory/` 对应卷目录下创建 `.md` 文件
2. 使用标准Frontmatter格式：
   ```yaml
   ---
   title: 文章标题
   description: 文章描述
   date: 2025-01-01
   category: 理论分类
   tags: [标签1, 标签2]
   ---
   ```
3. 提交Pull Request，等待审核

#### **添加案例研究**
1. 使用案例模板：`docs/case-studies/template.md`
2. 按分类放入对应目录
3. 包含：背景、IGT分析、数据、结论、建议

#### **开发交互工具**
1. 在 `apps/` 下创建新应用目录
2. 使用Vue 3 + Vite模板
3. 遵循统一设计规范
4. 提供完整文档和测试

### **构建与部署**
```bash
# 本地开发
npm run dev

# 构建生产版本
npm run build

# 本地预览
npm run preview

# 部署到Cloudflare Pages
# （自动通过GitHub Actions）
```
