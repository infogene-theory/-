# 信息基因论 IGT v20：基于"阴阳调和"的宇宙温控动力学

> **"宇宙没有永恒的结构，只有永恒的'破-立-衡'循环。演化的终局不是死寂，而是太极态的动态平衡。"**

## 🚀 快速开始

### 项目简介

本项目采用 **VitePress** 构建，致力于呈现一个**可计算、可验证、可执行**的统一演化框架。IGT v20 将哲学辩证法（阴阳）与硬核物理学（热力学、量子力学）融合，提供了一套跨越 14 个数量级的通用语言——**温度场**。

### 技术栈

- **前端架构**：VitePress + Vue 3 + Markdown
- **部署平台**：Cloudflare Pages
- **版本控制**：Git + GitHub + 语义化版本（v20.x）
- **开发环境**：Trae AI IDE + 本地开发服务器
- **内容管理**：Markdown结构化文档 + 理论模块化组件

### 安装和运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```
