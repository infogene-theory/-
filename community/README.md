# IGT社区贡献

本目录包含信息基因论IGT v20社区的贡献内容，包括讨论、答疑、工作坊组织等。

## 目录结构

### discussions/
社区成员的讨论和交流内容。

### faq/
常见问题解答。

### workshops/
工作坊组织和材料。

### contributions/
社区成员的贡献内容，包括案例、工具、可视化等。

### author-columns/
作者专栏，包括Damon专栏等。

## 参与方式

### 加入讨论

1. 浏览discussions目录下的讨论主题
2. 在对应的讨论文件中添加你的观点和评论
3. 提交更改并创建Pull Request

### 提问和答疑

1. 在faq目录下查找是否已有类似问题
2. 如果没有，创建新的FAQ文件
3. 提交问题并等待社区成员解答

### 组织工作坊

1. 在workshops目录下创建工作坊计划文件
2. 包含工作坊的主题、时间、地点、内容等信息
3. 提交计划并等待社区反馈
4. 组织工作坊并分享材料

### 贡献内容

1. 在contributions目录下创建贡献内容
2. 遵循相关的贡献指南和模板
3. 提交贡献并等待评审

## 贡献指南

### 讨论规范

- 尊重他人观点
- 保持理性和客观
- 提供有价值的内容
- 避免人身攻击和不当言论

### FAQ规范

- 问题描述清晰明确
- 回答内容准确全面
- 提供相关参考资料
- 保持格式统一

### 工作坊规范

- 主题明确，符合IGT理论
- 内容结构清晰，易于理解
- 提供实用的工具和方法
- 包含互动环节和实践活动

### 内容贡献规范

- 遵循相关目录的贡献指南
- 提供完整的文档和说明
- 确保内容的准确性和可靠性
- 遵循许可证要求
