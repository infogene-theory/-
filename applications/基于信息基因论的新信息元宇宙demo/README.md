# 新信息元宇宙平台

## 🌟 项目简介

新信息元宇宙平台是一个基于**信息基因论（IGT v7.0）**构建的去中心化共识知识网络。通过创新的频率相干性与RVSE演化机制（复制-变异-稳定-涌现），实现人类共识知识库的自主演化与AI系统的和谐共生；同时以可计算指标衡量并激励信息的价值（语义质量、稳定性、创新性、实用性）与信息载体的价值（数据源可信度、传播/计算/存储贡献度），形成"内容—载体—共识"的价值闭环。

### 🎯 核心使命
- **构建人类共识知识库**：建立三层金字塔结构的分布式知识网络
- **促进AI自演化**：确保AI发展方向与人类价值观保持一致
- **实现去中心化治理**：通过DAO机制实现真正的社区自治
- **推动知识文明进步**：为人类社会留下可持续演化的知识体系

## 🧬 理论基础：信息基因论

### 核心概念
**信息基因（IG）** = {数据结构, 语义内容, 频率特征}

### 关键公式（v7.0 三指标体系）
- **单尺度秩序度**：`O_freq = 1 - S_freq / log₂N`
- **尺度不变指数**：`I_STI = 1 - CV(O_freq^{(1)}, O_freq^{(2)}, ..., O_freq^{(M)})`
- **涌现速率**：`E_rate > 0.01/代`（用于判定跨尺度涌现）
- **价值评估**：`价值 = 知识价值 × 载体贡献系数(V_c)`

### RVSE演化循环（v7.0）
1. **复制（Replication）**：高`I_STI`结构获得复制优势
2. **变异（Variation）**：频率扰动与结构重组探索新模式
3. **稳定（Stabilization）**：多尺度筛选保留真秩序（`O_freq>0.6`且`I_STI>0.8`）
4. **涌现（Emergence）**：跨尺度频率跃迁与层级形成（`E_rate>0.01`）

### 价值评估体系
#### 知识价值评估
- **相干性评分**：基于频率相干性系数
- **稳定性评分**：基于历史验证数据  
- **创新性评分**：基于新颖度和影响力
- **实用性评分**：基于应用场景和用户反馈

#### 载体贡献系数(V_c)
- **来源可信度**：数据来源审计、可追溯性
- **传播影响力**：节点覆盖、带宽贡献
- **计算/存储贡献**：算力质押、存储时长
- **安全合规**：加密保护、隐私合规评分

**最终汇率**：`E_rate = Base_rate × Q_c × S_c × V_c`

### 核心算法
```javascript
// 频率相干性计算
function calculateCoherence(frequencies) {
    const n = frequencies.length;
    const sumFreq = frequencies.reduce((sum, freq) => sum + freq, 0);
    const sumFreqSquared = frequencies.reduce((sum, freq) => sum + freq * freq, 0);
    return Math.abs(sumFreq) / (n * sumFreqSquared);
}

// 多尺度验证
function multiScaleValidation(data, scales = [1, 2, 4, 8, 16]) {
    const cvValues = scales.map(scale => {
        const downsampled = data.filter((_, i) => i % scale === 0);
        const mean = downsampled.reduce((sum, val) => sum + val, 0) / downsampled.length;
        const variance = downsampled.reduce((sum, val) => sum + Math.pow(val - mean, 2), 0) / downsampled.length;
        return Math.sqrt(variance) / Math.abs(mean);
    });
    
    const avgCV = cvValues.reduce((sum, cv) => sum + cv, 0) / cvValues.length;
    return avgCV < 0.2; // 验证通过阈值
}
```
