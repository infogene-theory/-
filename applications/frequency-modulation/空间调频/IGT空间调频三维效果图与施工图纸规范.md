# IGT空间调频三维效果图与施工图纸规范

## 三维效果图设计规范

### 效果图设计目标
基于IGT-RVS机制的空间认知理论，通过三维可视化技术展现空间调频系统的安装效果、工作原理和视觉呈现，为项目实施提供直观的参考依据。

### 效果图分类

#### 场景效果图（Scene Renderings）
**睡眠空间效果图（SL）**
- **空间特征**：卧室环境，温馨舒适
- **光照条件**：暖色调，低照度
- **材质表现**：柔软材质，温暖质感
- **氛围营造**：宁静安详，有助于放松

**家庭空间效果图（FM）**
- **空间特征**：客厅/餐厅，开放互动
- **光照条件**：自然光+人工照明
- **材质表现**：天然材质，亲和力强
- **氛围营造**：和谐温馨，促进交流

**办公空间效果图（WR）**
- **空间特征**：工作区域，简洁高效
- **光照条件**：明亮均匀，无眩光
- **材质表现**：现代材质，科技感强
- **氛围营造**：专注高效，激发创意

#### 产品效果图（Product Renderings）
**设备类产品**
- 声波共振模块多角度展示
- 智能传感器细节表现
- 电子墨水屏界面展示
- 共振手环佩戴效果

**宣传类产品**
- 标语墙纸空间安装效果
- 不同VOI等级视觉对比
- 材质纹理细节表现
- 色彩搭配效果展示

#### 技术效果图（Technical Renderings）
**声波传播效果图**
- 频率波形可视化
- 声场分布模拟
- 共振效果展示
- 覆盖范围示意

**安装示意图**
- 安装位置标注
- 接线方式说明
- 固定方法展示
- 安全距离标识

### 三维建模规范

#### 建模精度要求
**几何精度**：
- 建筑模型：±5mm精度
- 产品模型：±1mm精度
- 装饰细节：±0.5mm精度

**材质精度**：
- 纹理分辨率：≥2048×2048px
- 法线贴图：16位高精度
- 反射贴图：HDR高动态范围

#### 光照设置规范
**全局光照**：
- 使用基于物理的渲染（PBR）
- 环境光遮蔽（AO）开启
- 全局照明（GI）精度：高
- 光线追踪反射：开启

**人工光源**：
- 色温设置：符合场景需求
- 强度设置：基于真实照度
- 阴影质量：高分辨率
- 光源类型：区域光源优先

#### 材质表现规范
**物理材质**：
- 基础色（Albedo）：基于真实色彩
- 金属度（Metallic）：物理准确
- 粗糙度（Roughness）：真实表现
- 法线（Normal）：细节丰富

**特殊材质**：
- 发光材质：强度适中
- 透明材质：折射率准确
- 次表面散射：皮肤/蜡质
- 各向异性：金属/布料

### 渲染输出规范

#### 分辨率设置
**主效果图**：
- 分辨率：4K（3840×2160px）
- 像素密度：300dpi
- 色彩深度：16位/通道
- 文件格式：PNG无损压缩

**细节展示图**：
- 分辨率：2K（1920×1080px）
- 像素密度：300dpi
- 色彩深度：8位/通道
- 文件格式：JPEG高质量

#### 色彩管理
**色彩空间**：
- 工作空间：sRGB IEC61966-2.1
- 输出色彩：Adobe RGB（1998）
- 位深度：16位/通道
- 伽马值：2.2

**色彩准确性**：
- 色彩偏差：ΔE≤2.0
- 灰阶平衡：中性灰不偏色
- 肤色表现：自然真实
- 材质色彩：物理准确

#### 后期处理规范
**基础调整**：
- 曝光校正：±0.3EV范围内
- 对比度调整：保持细节
- 色彩平衡：自然协调
- 锐化处理：适度增强

**特效处理**：
- 景深效果：符合视觉习惯
- 运动模糊：表现动态感
- 光晕效果：适度添加
- 粒子效果：增强氛围

## 施工图纸规范

### 图纸分类体系

#### 平面图（Floor Plans）
**空间布局图**
- 比例：1:50或1:100
- 内容：设备位置、安装尺寸
- 标注：详细尺寸、安全距离
- 符号：统一图例说明

**设备安装图**
- 比例：1:20或1:50
- 内容：安装高度、固定方式
- 标注：螺丝规格、承重能力
- 细节：接线位置、走线方式

#### 立面图（Elevations）
**墙面安装图**
- 比例：1:20或1:50
- 内容：垂直安装位置
- 标注：离地高度、间距尺寸
- 细节：固定点分布、载荷分布

**设备详图**
- 比例：1:5或1:10
- 内容：设备尺寸、接口位置
- 标注：精确尺寸、公差要求
- 细节：安装孔位、接线端子

#### 剖面图（Sections）
**结构剖面图**
- 比例：1:10或1:20
- 内容：内部结构、层次关系
- 标注：材料厚度、间隙尺寸
- 细节：连接方式、密封处理

**安装剖面图**
- 比例：1:5或1:10
- 内容：安装层次、固定结构
- 标注：螺丝深度、膨胀管规格
- 细节：防水处理、绝缘保护

### 图纸绘制规范

#### 线型标准
**可见轮廓线**：
- 线宽：0.5mm
- 线型：连续实线
- 用途：主要结构轮廓
- 颜色：黑色

**隐藏轮廓线**：
- 线宽：0.25mm
- 线型：虚线（2-1-2-1）
- 用途：被遮挡的结构
- 颜色：灰色（50%黑）

**尺寸标注线**：
- 线宽：0.18mm
- 线型：连续实线
- 用途：尺寸标注
- 颜色：黑色

**中心线**：
- 线宽：0.18mm
- 线型：点划线（10-2-2-2）
- 用途：对称中心线
- 颜色：红色

#### 标注规范
**尺寸标注**：
- 字体：Arial 3.5mm
- 单位：毫米（mm）
- 精度：±0.1mm
- 箭头：闭合填充箭头

**文字标注**：
- 字体：仿宋体 3.0mm
- 行距：1.5倍字高
- 对齐：左对齐
- 颜色：黑色

**符号标注**：
- 焊接符号：GB/T 324-2008
- 表面粗糙度：GB/T 131-2006
- 几何公差：GB/T 1182-2008

#### 图层管理
**图层命名规范**：
```
0-DEFPOINTS：默认层（不打印）
1-轮廓线：主要结构轮廓
2-隐藏线：被遮挡结构
3-中心线：对称中心线
4-尺寸线：尺寸标注
5-文字：文字说明
6-符号：技术符号
7-图框：图纸边框
8-标题栏：标题栏信息
```

### 图纸内容要求

#### 标题栏标准
**基本信息**：
- 项目名称：IGT空间调频项目
- 图纸名称：具体图纸名称
- 图纸编号：统一编码规则
- 设计阶段：方案/施工图/竣工图

**技术信息**：
- 比例：实际绘制比例
- 图幅：A0/A1/A2/A3/A4
- 版本号：版本控制标识
- 出图日期：YYYY-MM-DD格式

**责任信息**：
- 设计人：设计人员签名
- 校对人：校对人员签名
- 审核人：审核人员签名
- 批准人：批准人员签名

#### 技术要求
**通用技术要求**：
```
1. 本图尺寸单位除注明外均为毫米（mm）
2. 图中所注尺寸为成品尺寸，不含加工余量
3. 表面粗糙度未注明处为Ra 6.3μm
4. 焊接要求符合GB/T 985.1-2008标准
5. 涂装要求符合GB/T 9286-1998标准
```

**专用技术要求**：
```
1. 设备安装必须保证水平度≤2mm/m
2. 电气连接必须符合GB 50303-2015标准
3. 声学设备安装必须考虑振动隔离
4. 所有金属构件必须做防腐蚀处理
5. 隐蔽工程必须经验收后方可封闭
```

### 图纸输出规范

#### 文件格式
**矢量格式**：
- 主要格式：DWG 2018版本
- 备用格式：DXF 2013版本
- PDF格式：矢量输出，可搜索文字
- 包含内容：完整图层信息

**位图格式**：
- 分辨率：300dpi
- 色彩模式：黑白/灰度
- 文件格式：TIFF无损压缩
- 位深度：8位

#### 打印规范
**打印设置**：
- 打印机：大型绘图仪
- 纸张：80g/m²绘图纸
- 打印质量：高精度
- 线宽控制：按图层设置
