# IGT声波共振模块使用说明书

## 产品概述

IGT声波共振模块是空间调频系统的核心设备，基于信息基因论（IGT）和RVS（复制-变异-选择）机制设计，通过精准控制1-40Hz低频声波，实现与人体生物频率的共振，从而调节空间认知秩序。

## 技术参数

### 基础参数
- **产品型号**：IGT-RVS-EQ-XX-01（XX代表场景代码）
- **频率范围**：1-40Hz（±5%精度）
- **声压输出**：≤60dB（1米处测量）
- **总谐波失真**：≤3%
- **频率响应**：±2dB平坦度

### 电气参数
- **工作电压**：3.0-5.0V DC
- **工作电流**：50-150mA
- **功率消耗**：0.15-0.75W
- **电源接口**：Micro-USB/弹簧夹
- **电池寿命**：连续工作≥72小时（3V电池）

### 环境参数
- **工作温度**：-10℃ ~ +45℃
- **存储温度**：-20℃ ~ +60℃
- **工作湿度**：5% ~ 95%RH（无冷凝）
- **防护等级**：IP32（室内使用）

## 安装步骤

### 第一步：环境评估
1. **空间大小确认**：
   - 小空间（≤20㎡）：1个模块
   - 中空间（20-50㎡）：2个模块
   - 大空间（≥50㎡）：3个模块

2. **安装位置选择**：
   - 高度：距离地面1.5-1.8米
   - 方向：朝向主要活动区域
   - 距离：距离墙面≥30cm
   - 避免：空调出风口、直射阳光

### 第二步：固定安装
1. **清洁安装面**：
   - 用酒精棉清洁墙面
   - 确保表面干燥无尘
   - 等待2分钟让酒精挥发

2. **粘贴固定**：
   - 撕下3M胶贴保护膜
   - 按压在模块背面中心
   - 对准位置用力按压30秒
   - 等待24小时达到最大粘性

### 第三步：电源连接
1. **电池安装**：
   - 打开电池盒盖
   - 按正负极标识装入AA电池
   - 确保电池接触良好
   - 关闭电池盒盖

2. **电源检查**：
   - 观察电源指示灯（红色常亮）
   - 确认设备正常启动
   - 检查电池电量状态

### 第四步：频率设置
1. **场景选择**：
   ```
   睡眠场景（SL）：5-7Hz
   家庭场景（FM）：9-11Hz
   办公场景（WR）：16-24Hz
   会议场景（MT）：28Hz
   教育场景（ED）：15Hz
   康复场景（RT）：3Hz
   ```

2. **调节方法**：
   - 短按设置键进入调节模式
   - 按+/-键调节频率值
   - 长按设置键保存设置
   - 自动记忆上次设置

## 使用操作

### 基本操作
1. **开机/关机**：
   - 长按电源键3秒开机
   - 长按电源键3秒关机
   - 自动关机：无操作30分钟后

2. **频率调节**：
   - 按+键：频率增加1Hz
   - 按-键：频率减少1Hz
   - 长按：快速连续调节

3. **音量控制**：
   - 设备自动调节最佳音量
   - 手动调节：同时按+/-键
   - 建议音量：≤60dB（舒适范围）

### 高级功能
1. **定时功能**：
   - 支持1-8小时定时
   - 自动关机保护
   - 记忆功能保存设置

2. **同步功能**（多设备）：
   - 主机设置自动同步
   - 支持最多8台设备
   - 同步延迟≤100ms

3. **智能模式**（选配）：
   - 传感器自动调节
   - APP远程控制
   - 数据统计分析

## 故障排除

### 常见问题
1. **无法开机**：
   - 检查电池安装是否正确
   - 确认电池有电
   - 检查电源接触点

2. **无声音输出**：
   - 检查音量设置
   - 确认频率设置正确
   - 检查扬声器连接

3. **频率不稳定**：
   - 检查环境温度
   - 确认电源电压稳定
   - 避免强电磁干扰

4. **异常噪音**：
   - 检查固定是否牢固
   - 确认安装面平整
   - 检查内部零件松动

### 故障代码
```
E01：电池电量低
E02：温度传感器故障
E03：频率发生器故障
E04：内存错误
E05：通信故障
```
